Danke für die Benutzung des Sitemap Modules. Das Module wird automatisch eine Sitemap für Ihre Website erstellen.

  • Nach der Installation des Modules, bearbeiten Sie bitte in den Einstellungen die Nachricht oder löschen Sie diese.
  • Wenn Sie die Funktion der Google Sitemaps benutzen wollen,kopieren Sie die Datei von modules/sitemap/xml_google.php in Ihr Basisverzeichnis (root) Ihres ImpressCMS, und registrieren Sie anschließend die URL ICMS_URL/xml_google.php in den Google Webmaster Tools.
